# Eyeson-core-front
코어 프론트 프로젝트 소스입니다.

## Installing
1. npm install
2. npm start

## Functions
1. 디바이스 세팅 
2. 화상 연결
3. 개인권한 제어
    - 개인 카메라 , 마이크 조작 가능
4. 타인권환 제어
    - 타인 카메라 , 마이크 조작 가능
5. 채팅 기능
6. 타인에게 알림 기능
    - 알림 후 채팅에 상황 메세지 전달
7. 강퇴 기능
    - 일부 사용자 방에서 내보내기 기능
8. 에러 및 유효성 체크
    - 존재하지 않는 방에 입장할경우
    - 방 입장 이후 방이 삭제된 경우
    - 존재하지 않는 유저인 경우
    - 유효하지 않는 토큰
    - 만료된 토큰
    - 중복 로그인
